Class Notes: CSS Selectors

  • Selector is used to target specific HTML elements to apply CSS styling.
  • Syntax:
    • selector { property: value; }
  • Property: The CSS property that you want to apply on the targeted HTML element.

Types of Selectors:

  1. Simple Selectors: Used to select elements based on basic criteria like name, id, class, etc.

    1. Element/Tag Selector: Selects all elements of a specific type.
      Example: p { color: red; }

    2. ID Selector (#): Selects an element with a specific id. It has higher priority than class.
      Example: #header { font-size: 20px; }

    3. Class Selector (.): Selects all elements with a specific class.
      Example: .
      menu { background-color: yellow; }

    4. Universal Selector (*): Selects all elements on the page.
      Example: * { margin: 0; }

    5. Grouping Selector (,): Applies same style to multiple selectors.
      Example: h1, h2, p { font-family: Arial; }

  2. Combinator Selectors: Used to define relationships between elements.

    1. Child Selector (>): Selects only direct children.
      Example: div > p(selects only p elements that are direct children of div)

    2. Descendant Selector (space): Selects all descendants.
      Example: div p (selects all p elements inside div, no matter how deep)

    3. Adjacent Sibling Selector (+): Selects the element immediately after another.
      Example: h1 + p

    4. General Sibling Selector (~): Selects all siblings after the specified element.
      Example: h1 ~ p

  3. Pseudo-Class Selectors (:) - Used to define the special state of an element.

    1. :link – Unvisited link
    2. :visited – Visited link
    3. :active – Link when it is being clicked
    4. :focus – Element when it gets focus (like input box)
    5. :hover – Element when mouse is over it
    6. :first-child – Selects the first child of a parent
    7. :last-child – Selects the last child of a parent
    8. :nth-child(n) – Selects the nth child of a parent
    9. :nth-last-child(n) – Selects the nth child from end
    10. :first-of-type – First of its type among siblings
    11. :last-of-type – Last of its type among siblings
    12. :nth-of-type(n) – nth element of its type
    13. :nth-last-of-type(n) – nth from end of its type

    14. :is() – Selects if element matches any of given selectors.
      Example: :is(h1, h2) { color: red; }

    15. :has() – Selects if an element contains a certain child (Not supported in all browsers yet).

    16. :where() – Similar to :is(), but has no specificity.

    17. :not() – Selects everything except the specified selector.
      Example: div:not(.active)

  4. Pseudo-Element Selectors (::) - Used to style specific parts of elements.

    1. ::first-letter – Styles the first letter of a text
    2. ::first-line – Styles the first line of a text
    3. ::selection – Styles the part of an element that is selected
    4. ::marker – Styles the marker of list items (like bullet)
    5. ::before – Inserts content before the element
    6. ::after – Inserts content after the element
    7. ::placeholder – Styles the placeholder text inside form inputs

  5. Attribute Selectors: Used to select elements with specific attributes.

    1. input[type] – Selects input elements with any type attribute
    2. input[type="text"] – Selects input elements with type="text"
    3. a[href^="https"] – Selects elements where href starts with "https"
    4. a[href$=".pdf"] – Selects elements where href ends with ".pdf"

    5. [title~="flower"] – Selects elements whose title contains word "flower"

    6. [class*="button"] – Selects elements whose class contains "button" anywhere
