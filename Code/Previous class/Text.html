CSS Text Elements – Assignment Task List

  1. Font Properties
    1. Apply different font-family like Arial, 'Times New Roman', etc.
    2. Use font-size in px, em, rem.
    3. Style text using font-style: normal, italic, oblique.
    4. Set font-weight: normal, bold, 300, 700, etc.
    5. Try font-variant: small-caps.
    6. Adjust line-height to improve readability.
    7. Use letter-spacing and word-spacing to space out letters/words.
  2. Text Alignment & Indentation
    1. Align text: text-align: left, center, right, justify.
    2. Indent paragraph using text-indent.
  3. Text Decoration
    1. Underline, overline, strikethrough using text-decoration.
    2. Change decoration color, style, thickness.
  4. Text Transform
    1. Use text-transform: uppercase, lowercase, capitalize.
  5. Text Shadow
    1. Add shadow using text-shadow: e.g., 2px 2px 5px gray.
  6. White Space & Wrapping
    1. Control wrapping using white-space: normal, nowrap, pre.
    2. Break long words using overflow-wrap and word-break.
  7. Direction & Writing Mode
    1. Try direction: rtl and unicode-bidi: bidi-override.
    2. Apply writing-mode: vertical-rl for vertical layout.
  8. List Styling
    1. Customize list using list-style-type: circle, square, decimal.
    2. Try list-style-position: inside, outside.
    3. Use list-style-image to replace bullets with an image.
  9. Quotes Styling
    1. Use blockquote and q tags with quotes property.
    2. Add custom quote marks using ::before and ::after.
  10. Highlighting and Selection
    1. Use ::selection to change text and background color when selected.
  11. Advanced (Bonus)
    1. Import and use fonts from Google Fonts.
    2. Apply clamp() to make font-size responsive.
    3. Use transition on text color or size on hover.
    4. Add media queries to change text styles on small screens.
