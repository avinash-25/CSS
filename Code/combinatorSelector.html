Class Notes (Selectors)

  • Selector is used to target the HTML element so that we can apply property.
    • Selector{ property : Value }
  • Property : The property we want to apply on html element that we have targeted.

Types of selector :-

  1. Simple Selector :

    1. Tagname/ Element
    2. ID Selector. (#)
    3. Class Selection (.)
    4. Universal selector. (*)
    5. Grooping Selector. (,)

  2. Combinator Selector :

    1. Direct child selection (>)
    2. Descendent selector (_)
    3. Immediate selector (+)
    4. General Sibling selector (~)

  3. Class selector :

    1. : link
    2. : active
    3. : visited
    4. : focus
    5. : hover
    6. : first-child
    7. : last-child
    8. : nth child
    9. : nth-last-child()
    10. : first-of-type
    11. : last-of-type
    12. : nth-of-type()
    13. : nth-last-of-type()
    14. : is()
    15. : hos()
    16. : where()
    17. : not()

  4. Pseudo Element selector (::)

    1. :: first letter
    2. :: first line.
    3. :: selection
    4. :: marker
    5. :: before
    6. :: after
    7. :: Placeholder

  5. Attribute Selector

    1. Tagname [Attribute]
    2. Tagname [Attribute = 'value']
    3. Tagname [Attribute ^= "Prefix value"]
    4. Tagname [Attribute $= "Suffix value"]
    5. Tagname [Attribute ~= "mid value"]
    6. Tagname [Attribute *= "any value"]


Priority

  1. ! important
  2. ID Selector (#)
  3. Class Selector (.)
  4. Tagname/Element selection
  5. Universal Selector (*)
