Grid Tracks Explanation

Grid Tracks: The space between any two consecutive grid lines is called a grid track. It is basically the space or the area between the lines.

Column Tracks: Vertical spaces between vertical grid lines
Row Tracks: Horizontal spaces between horizontal grid lines

Grid Cells Explanation

Grid Cells: Grid cells are the space present between any four intersecting grid lines. From the table analogy every table cell can be thought of as a grid cell too. It is the smallest unit in the CSS grid.

Each cell is formed by the intersection of one row track and one column track.
