Exercise 4: Responsive Grid Challenge

🎯 Objective:

Create a responsive product grid that adapts to different screen sizes using CSS Grid.

📋 Requirements:

  • Desktop (>768px): 4 columns
  • Tablet (481-768px): 3 columns
  • Mobile (≤480px): 2 columns
  • Use auto-fit and minmax()
  • Items should have minimum width of 200px
  • Add 20px gap between items
  • Create 12 product cards
/* Your responsive CSS here */ .exercise4 { display: grid; /* Use auto-fit and minmax for responsiveness */ } @media (max-width: 768px) { /* Tablet styles */ } @media (max-width: 480px) { /* Mobile styles */ }
💡 Hint: Try grid-template-columns: repeat(auto-fit, minmax(200px, 1fr))

Exercise 5: Named Grid Areas

🎯 Objective:

Create a dashboard layout using named grid areas for better code readability.

📋 Requirements:

  • Use grid-template-areas to define layout
  • Create areas: "nav", "sidebar", "content", "ads", "footer"
  • Nav should span full width at top
  • Sidebar, content, and ads in middle row
  • Footer spans full width at bottom
  • Make it visually appealing with different colors
/* Your named areas CSS here */ .exercise5 { display: grid; grid-template-areas: "nav nav nav" "sidebar content ads" "footer footer footer"; /* Add more properties */ } .nav { grid-area: nav; } /* Define other areas */
🔥 Pro Tip: Named grid areas make your code more readable and maintainable!
