CSS Pseudo-Class Selector Practice Questions

  1. Link Pseudo-Classes
    1. Create a navigation menu with 4 links.
    2. Style unvisited links (:link) with a blue color and no underline.
    3. Style visited links (:visited) with a purple color.
    4. On hover (:hover), add a red underline and a slight scale increase using transform: scale(1.1) with a smooth transition.
    5. When clicked (:active), change the link color to green.
  2. Focus and Focus-Within
    1. Build a form with 2 text inputs and a submit button.
    2. Style focused inputs (:focus) with a 2px orange outline and a light yellow background.
    3. Apply a subtle blue border to the entire form when any input is focused using :focus-within.
  3. Form-Related Pseudo-Classes
    1. Create a form with a required email input and a disabled text input.
    2. Style required inputs (:required) with a red border.
    3. Style the disabled input (:disabled) with a gray background and reduced opacity.
    4. Style valid inputs (:valid) with a green border.
    5. Style invalid inputs (:invalid) with a red border and a subtle shake animation using @keyframes.
  4. Structural Pseudo-Classes (Child)
    1. Write a section with 5 paragraphs.
    2. Style the first paragraph (:first-child) with bold text and a larger font size.
    3. Style the last paragraph (:last-child) with italic text and a gray background.
    4. Apply a blue background to the second paragraph (:nth-child(2)).
    5. Apply a green background to the second-to-last paragraph (:nth-last-child(2)).
  5. Structural Pseudo-Classes (Type)
    1. Create an unordered list with 6 items.
    2. Style the first list item (:first-of-type) with a red border.
    3. Style the last list item (:last-of-type) with a bold font and a yellow background.
    4. Apply alternating background colors using :nth-of-type(even) and :nth-of-type(odd).
  6. Table Pseudo-Classes
    1. Build a table with 5 rows and 3 columns (e.g., Name, Age, City).
    2. Style odd-numbered rows (:nth-child(odd)) with a light blue background.
    3. Style even-numbered rows (:nth-child(even)) with a light green background.
    4. Apply bold font to the second column (:nth-of-type(2)).
  7. Content-Based Pseudo-Classes
    1. In a section with 3 paragraphs, style <strong> elements within the first paragraph using :where(p:first-child) with a red color.
    2. Style all links that do not have a specific class (e.g., external) using :not(.external) with an underline and teal color.
  8. Interactive Pseudo-Classes
    1. Create a button.
    2. Style it when enabled (:enabled) with a green background and white text.
    3. Style it when disabled (:disabled) with a gray background and a not-allowed cursor.
    4. Add a hover effect (:hover) to the enabled button that increases brightness using filter: brightness(1.2).
  9. Negation and Empty Pseudo-Classes
    1. Create a form with a textarea.
    2. Style an empty textarea (:empty) with a light red background and placeholder-like border.
    3. Style all paragraphs except the first one (:not(:first-child)) with a smaller font size and a left border.
  10. Advanced Structural Pseudo-Classes
    1. In a table with 4 columns, style the second column (:nth-of-type(2)) with centered text and a light yellow background.
    2. Style the last column (:nth-last-of-type(1)) with uppercase text and a blue border.
  11. Combined Pseudo-Classes (Bonus)
    1. Style unvisited navigation links on hover (:hover:not(:visited)) with a yellow background and a smooth fade-in effect.
    2. Style required inputs that are both focused and valid (:focus:required:valid) with a green glow (box-shadow) and a checkmark background image.
    3. Style even-numbered list items that are not the last item (:nth-of-type(even):not(:last-of-type)) with a dashed border.
